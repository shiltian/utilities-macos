# Utilities

A native macOS utilities app built with SwiftUI.

## Tools

### Markdown Preview

Convert Markdown to HTML with real-time preview. Supports multiple output formats (preview, raw HTML, HTML with CSS) and auto-saves your work.

## Requirements

- macOS 14.0 (Sonoma) or later
- Xcode 15.0 or later (for building from source)
